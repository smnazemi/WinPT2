This is a sample project to demonstrate the functionality of IPCFramework. 

Please make sure that Remote PT is listening at port no. 39000 or use the port number in test application on which remote PT is listening. 

Steps need to be followed:

-	Click on the “start framework” button to start the framework. 

-	Click on “Test FW” button to test the sample functionality of the framework (which displays port name of Router0 in this case). 

-	Click on “Subscribe” button to subscribe for the events of your choice.(powerChanged is the event Subscribed in the example). 

-	Click on “Unsubscribe” button to unsubscribe the event that was already subscribed.
